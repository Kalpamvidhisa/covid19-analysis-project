# 📊 COVID-19 Time Series Data Analysis

## 🚀 Project Overview
This project focuses on analyzing global COVID-19 time series data to understand trends, peaks, and spread patterns across different countries. The analysis is performed using Python in Google Colab.

---

## 🎯 Objectives
- Load and process country-wise COVID-19 time series data  
- Compute daily and weekly case counts  
- Apply rolling averages to smooth fluctuations  
- Compare trends across multiple countries  
- Detect peak infection periods  
- Estimate basic reproduction rate (Rt)  

---

## 📂 Dataset
- Source: Johns Hopkins University COVID-19 Dataset  
- Format: Time series (global confirmed cases)  
- Data includes country-wise daily cumulative cases  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Plotly (for interactive visualization)  
- Google Colab  
- GitHub  

---

## 📊 Analysis Performed

### 1. Data Preprocessing
- Cleaned and structured dataset  
- Aggregated data country-wise  
- Converted date columns into time series format  

### 2. Daily & Weekly Cases
- Computed daily new cases using differencing  
- Calculated weekly trends for better insights  

### 3. Rolling Average
- Applied 7-day rolling average to smooth noisy data  

### 4. Country Comparison
- Compared trends across countries such as India, US, and Brazil  

### 5. Peak Detection
- Identified peak infection dates for each country  

### 6. Reproduction Rate (Rt)
- Estimated approximate reproduction rate  
- Analyzed spread patterns (Rt > 1 indicates growth, Rt < 1 indicates decline)  

---

## 📈 Key Insights
- Multiple waves of COVID-19 observed across countries  
- Significant peaks occurred at different time periods globally  
- Rolling averages provided clearer trend visualization  
- Rt values highlighted phases of rapid spread and control
